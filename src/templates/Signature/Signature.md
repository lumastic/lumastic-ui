#### `Signature` takes two children: a seal and text.

#### User Signature

```jsx
import { Avatar } from "../../components/Avatar";
import { Type } from "../../components/Type";
<Signature>
  <Avatar />
  <Type>User Signature</Type>
</Signature>;
```

#### Icon Signature

```jsx
import { Sparks } from "../../icons/Sparks";
import { Type } from "../../components/Type";
<Signature>
  <Sparks />
  <Type>Spark Signature</Type>
</Signature>;
```
