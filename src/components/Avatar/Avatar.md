#### Basic Usage

```jsx
<Avatar />
```

#### `size` prop differentiates importance

#### `setSize` prop can be used to override default sizes

```jsx padded
<Avatar setSize={"5rem"} />
<Avatar size={"large"} />
<Avatar />
<Avatar size={"small"} />
```

#### `shadow` prop isolates avatar from its context on the page

```jsx padded
<Avatar size={"large"} shadow />
```
