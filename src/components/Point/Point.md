#### Basic Usage

```jsx
<Point />
```

#### `color`

```jsx padded
<Point />
<Point color={"light"}/>
<Point color={"green"}/>
<Point color={"red"}/>
<Point color={"grey"}/>
```

#### `withBorder`

```jsx padded
<Point withBorder />
<Point color={"light"} withBorder/>
```

#### `size` "normal" or "small"

```jsx padded
<Point />
<Point size={"small"}/>
```
